# 🤖 Machine Learning Projects Collection

A comprehensive collection of **Machine Learning** and **Natural Language Processing (NLP)** projects built using **Python** and **Jupyter Notebook**. This repository demonstrates end-to-end machine learning workflows, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

Whether you're a beginner learning Machine Learning or a recruiter reviewing my work, this repository showcases practical implementations of real-world ML problems.

---

## 📂 Projects Included

### 🏠 House Price Prediction
- Predict house prices using regression algorithms.
- Data preprocessing and feature engineering.
- Model evaluation using MAE, MSE, RMSE, and R² Score.

### 🚢 Titanic Survival Prediction
- Binary classification to predict passenger survival.
- Handling missing values and categorical features.
- Feature engineering and classification models.

### 🌸 Iris Flower Classification
- Classify Iris flowers into three species.
- Data visualization and feature analysis.
- Classification using supervised learning algorithms.

### 💰 Insurance Cost Prediction
- Predict medical insurance charges based on customer attributes.
- Regression model implementation.
- Performance comparison of different algorithms.

### 🚗 Ford Car Price Prediction
- Predict used Ford car prices using regression techniques.
- Data cleaning and feature selection.
- Model optimization and evaluation.

### 📝 Natural Language Processing (NLP)
- Text preprocessing and cleaning
- Tokenization
- Stopword Removal
- Stemming & Lemmatization
- TF-IDF Vectorization
- Bag of Words (BoW)
- Text Classification

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Machine-Learning-Projects.git
```

2. Navigate to the project folder

```bash
cd Machine-Learning-Projects
```

3. Launch Jupyter Notebook

```bash
jupyter notebook
```

4. Open any notebook and run the cells.
